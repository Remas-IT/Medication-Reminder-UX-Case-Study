## 5. Research-Based Analysis

The findings align with established gerontechnology literature:

- Reduced contrast negatively impacts elderly readability
- Concrete and familiar icons improve recognition
- Clear labeling reduces cognitive load
- Large typography enhances usability

References used are included in the full report.

## 6. Conclusion & Future Improvements (My Contribution)

Based on evaluation results, the following improvements were proposed:

1. Increase font size and color contrast for all essential interactive elements.
2. Replace abstract navigation icons with universally recognized symbols.
3. Redesign confirmation screen with explicit labels (e.g., "Return to Home").
4. Introduce voice-assisted navigation for accessibility.
5. Conduct broader usability testing with larger elderly sample.
6. Develop high-fidelity and functional prototype for real-world validation.

These improvements aim to enhance accessibility, reduce cognitive strain, and increase task success rates.
